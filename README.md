# 🚀 WorkPulse – Task Management & Productivity Analytics System

## 📌 Overview
WorkPulse is a role-based task management system designed to streamline task assignment, tracking, and performance monitoring for teams.

## ✨ Features
- 🔐 Role-based authentication (Manager & Employee)
- 📋 Task creation, assignment, and tracking
- 🔁 Recurring task automation (daily/weekly/monthly)
- ⏱ Overdue detection and priority management
- 📊 Productivity analytics dashboard
- 📝 Activity logging system

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- REST APIs

## 📊 Analytics Capabilities
- Task completion rate
- Productivity score calculation
- Priority distribution
- Category-wise performance tracking
- Upcoming workload insights
